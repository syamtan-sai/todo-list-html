# To-Do List – HTML ✅

A simple **to-do list web app** built using **only HTML**.  
It allows users to **add tasks** and **delete tasks** with a minimal interface.

---

## 📌 Features
- Add new tasks
- Delete tasks
- Minimal, beginner-friendly code

---

## 🛠️ Tech Stack
- **HTML5** – structure & content
